# 🚀 Founder Simulator: The Global Startup Engine

FounderSim is a high-fidelity, multi-agent orchestration platform designed to stress-test startup visions with brutal honesty. Powered by **Sarvam AI (105B)** and **Tavily Search**, it deploys a specialized crew of AI agents to analyze market size, competitive landscapes, investor concerns, and customer skepticism across the globe.

![FounderSim Dashboard](https://images.unsplash.com/photo-1551288049-bebda4e38f71?auto=format&fit=crop&q=80&w=2070)

## 🌟 Key Features

- **Global Market Intelligence**: Real-time analysis of startups in the USA, India, SE Asia, and Europe.
- **Agentic Stress-Testing**: A crew of 8+ specialized agents (VC Partners, Skeptical Customers, Growth Strategists) scrutinize every aspect of your idea.
- **10-Year Trajectory**: Predictive modeling on how your startup and the competition will evolve over a decade.
- **Interactive Visualizations**: Data-rich reports with survival probability gauges, market maturity charts, and competitive maps.
- **Premium UI**: A sleek, Royal Indigo dark-themed interface built with Next.js, Tailwind CSS, and Framer Motion.

## 🧠 The Agent Crew

1.  **Global Market Analyst**: Scans world markets and identifies regional nuances.
2.  **Tier-1 VC Partner**: Asks the hard questions that kill 90% of pitches.
3.  **Competitor Mapper**: Finds direct and stealth competitors globally.
4.  **Customer Skeptic**: Channels the voice of the busy, risk-averse user.
5.  **Distribution Strategist**: Maps the path from 0 to 10k customers.
6.  **Moat Analyzer**: Evaluates long-term defensibility.
7.  **UX & Brand Director**: Defines the visual identity and design language.
8.  **Strategic Synthesizer**: Compiles all findings into a cohesive, actionable report.

## 🛠️ Technology Stack

- **Frontend**: Next.js 15, React 18, Tailwind CSS, Framer Motion, Recharts, Lucide Icons.
- **Backend**: FastAPI, Python 3.10+, CrewAI (Agentic Framework).
- **AI Models**: Sarvam AI (105B) for reasoning, Tavily for real-time web research.

## 🚀 Quick Start

### 1. Environment Setup
Configure your API keys in `backend/.env`:
```env
SARVAM_API_KEY=your_key
TAVILY_API_KEY=your_key
```

### 2. Run Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 3. Run Frontend
```bash
cd frontend
npm install
npm run dev
```

Visit `http://localhost:3000` to start your analysis.
